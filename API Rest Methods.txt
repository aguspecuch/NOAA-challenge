API Rest Methods

POST /boyas     		=> permite la creación boyas.
      RequestBody: 
      {
                     “longitudInstalacion”: 34.8877,
                     “latitudInstalacion”: 78.230
               }

GET /boyas 		=> devuelve las boyas SIN las muestras.

GET /boyas/{id} 	=> devuelve la información de una boya en particular (SIN las muestras).

PUT /boyas/{id} 	=> actualiza SOLO el color de luz de la boya. 
RequestBody:
               {
                       “color”: “ROJO”
               }

POST /muestras 	=> registra una muestra.
               RequestBody 
                            {
                                      “boyaId”: 32,
                                      “horario”: “2020-08-08T22:25:30”, 
                                      “matricula”: “99D9AAK”,
                                      “latitud”: -17.44681203,
                                      “longitud”: -113.16478854,
                                      “alturaNivelDelMar”: 50
                            }
                 Respuesta Esperada (JSON):
                            {
                                      “id”: 25,
                                      “color”: “Un Color”
                            }

Nota: Si cuando se carga una muestra nueva en una boya, la alturaNivelDelMar es de MENOS de
-50(menos 50) o de MÁS de +50 (más 50), deberá devolver “AMARILLO” en el color.
En el caso de que sea menor a -100 o mayor a +100 el color deberá ser ROJO.
Si no, el color deberá devolver es VERDE.

GET /muestras/boyas/{idBoya} 		=> devuelve la lista de muestras de una boya.

DELETE /muestras/{id}: 			=> Reseteara el color de la luz de la boya a “AZUL” a partir de una  muestra específica.
